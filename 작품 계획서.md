# 게임 타이틀 : 강의실 지각 러너 - 김성환, 202021469
Kv
## 기획 의도
이 게임은 대학생이 가장 공감하기 쉬운 상황인 “지각”을 소재로 만든 가벼운 캐주얼 게임이다. 복잡한 스토리보다 직관적인 조작, 빠른 판정, 짧은 플레이타임에 집중해서 단독 개발이 가능하다.
### 핵심 재미
- 긴장감 : 시간 제한 때문에 계속 급하게 뛰게 됨
- 회피 쾌감: 장애물을 피할 때의 즉각적인 반응
- 성취감: 기록 갱신, 클리어, 연속 성공 보상
### 차별화 포인트
- 학교 생활 공감형 콘셉트
- 단순한 러너지만 장애물을 “지각 방해 요소”로 바꿔서 재미를 줌
- 웹게임이라 누구나 바로 실행 가능
### 게임 핵심 루프KvK
1. 게임 시작
2. 플레이어가 자동 또는 반자동으로 전진
3. 장애물 등장
4. 점프 / 슬라이드 / 이동으로 회피
5. 아이템 획득 또는 시간 감속
6. 강의실 도착 또는 시간 초과
7. 점수 확인 후 재도전
### 조작 방식
- 이동: ← → 또는 A / D
- 점프: Space
- 슬라이드: ↓ 또는 S
- 재시작 : R
### 장애물 예시 (학교 풍경 컨셉)
- 사람 무리
- 계단
- 청소카트
- 닫히는 문
- 쏟아진 물
- 늦은 출서체크 알림
### 아이템 예시
- 에너지 드링크 : 잠깐 속도 증가
- 지름길 지도 : 잠시 장애물 예고 표시
- 아침 커피 : 점프 높이 증가 또는 대시 1회
- 시간표 알림 : 남은 시간 3초 추가
### 스테이지 구성
#### 1스테이지 : 복도
- 기본 장애물만 등장
- 조작법 익히는 구간
#### 2스테이지 : 계단
- 위아래 판정이 필요한 장애물 추가
- 속도 상승
#### 3스테이지 : 본관 앞
- 사람 무리, 문, 물체 등 복합 장애물
- 클리어 직전 난이도 상승
### UI 구성 (최대한 깔끔하게)
- 시작 화면
- 게임 플레이 화면
- 남은 시간 표시
- 점수 표시
- 현재 구간 표시
- 게임 오버 / 클리어 화면
### 실패 조건
- 시간 초과
- 장애물 충돌 3회
- 채력 0
### 성공 조건
- 제한 시간 안에 강의실 도착
### 난이도 설계
- 초반: 장애물 간격 넓게
- 중반: 장애물 속도 증가
- 후반: 점프와 슬라이드를 동시에 요구
- 마지막: 짧은 구간에서 연속 회피
